# vue-bookArtifact

> 仿追书神器vue项目

项目使用的是 `vue + vue-roter + axios + sass`构建的项目。后期如果需要更换成`vue-electron`直接修改即可。
因为`vue`自带了[autoprefixer](https://www.jianshu.com/p/f5b0b92e6b0f)这里就不再详细的补充了。
## Build Setup
  
``` bash
# 安装依赖
npm install

# 默认运行8080端口
npm run dev

# 打包
npm run build
```
